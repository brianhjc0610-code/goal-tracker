☁️ 云同步设置

连接 Supabase 后，数据会自动在所有设备间同步。只需配置一次。

第一步：创建 Supabase 项目

1. 访问 supabase.com 注册免费账号
2. 点击 "New Project"，填写名称和密码
3. 等待项目创建完成（约2分钟）

第二步：创建数据表

进入项目 → 左侧点 "SQL Editor" → 粘贴以下代码并运行： 
create table goal_data (
  id text primary key default 'main',
  data jsonb not null,
  updated_at timestamptz default now()
);

alter table goal_data enable row level security;

create policy "Allow all" on goal_data
  for all using (true) with check (true);

insert into goal_data (id, data) values ('main', '{}');
